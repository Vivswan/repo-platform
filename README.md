# repo-platform build branch (staging)

This branch is GENERATED by the build-branches workflow on `main` - do not
commit to it or open pull requests against it (they are closed automatically).
Template sources live in `templates/` on `main`.

- `staging` is rebuilt from the latest `main` commit on every push.
- `latest` is rebuilt from the latest release and tagged `templates/vX.Y.Z`.

Consume with copier, e.g.:

    copier copy gh:Vivswan/repo-platform . --vcs-ref templates/vX.Y.Z   # latest
    copier copy gh:Vivswan/repo-platform . --vcs-ref staging            # staging
